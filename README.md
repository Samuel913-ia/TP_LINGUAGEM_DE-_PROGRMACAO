Este projeto consiste no desenvolvimento de um sistema de autenticação básico em Python, 
utilizando decoradores para controle de acesso e hash de senhas com salt para garantir maior segurança. 
O sistema permite o registo e autenticação de utilizadores, 
mantendo um estado de sessão ativo e protegendo funções com base nas permissões (roles) atribuídas a cada utilizador, como “admin” ou “user”.
